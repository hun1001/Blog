Conditional Statements if else

조건문 if else

// bool 값
boolean b = true;
if(b)
{
    // b == true
}
else
{
    // b == false
}

실습 예제

Operator ||, &&

조건이 2개일 때

boolean a = true;
boolean b = false;

// or 연산: a와 b 중 하나라도 true라면 true
boolean result = a || b;

// and 연산: 둘 다 true일때만 true
result = a && b;

else if

    }

for & for

이중 반복문
